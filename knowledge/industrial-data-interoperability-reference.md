# 工业数据互操作参考

## OPC UA可借鉴能力

- 设备、机器和企业系统之间的互操作
- 信息模型和语义化对象
- 实时数据、历史数据和事件
- 订阅、读写和方法调用
- 告警与条件
- 认证、加密、签名和审计
- 多供应商设备和平台的安全连接

## 设计边界

- OPC UA是互操作参考，不是所有项目必须采用的技术协议。
- 设计时仍需确认现场协议、边缘平台、网络边界、数据主责和系统能力。
- 设备数据进入LIMS或MES前，应明确采集时间、质量状态、单位、校验和人工确认规则。
- 实时采集数据不默认等于最终检测结果或质量判定。
