# Explanation Content Standard

说明必须包含：

## 通用入口规则
- 说明 icon 必须统一样式
- 说明 icon 只表达说明能力
- 说明 icon 不得与业务功能按钮混合
- 同类重复组件只说明一次
- 组件实例不得重复说明
- 说明内容优先按模块、页面、区域、组件聚合
