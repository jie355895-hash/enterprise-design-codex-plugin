# Bundled Industrial Product Design Reference

本目录包含可公开分发的通用业务参考，用于帮助产品设计任务理解 LIMS、MES、QMS、IIoT、设备、质量、试验和工业 SaaS 场景。

知识内容只作为业务参考，不替代当前项目需求、用户确认内容、项目权限配置或插件强制规则。公开包不包含客户名称、个人信息、内部路径、原始业务文档或私有项目资料。

## 使用方式

- 先根据 `INDEX.md` 选择相关业务领域，再读取必要文件；
- 在项目控制目录记录采用内容、未采用内容、知识版本、假设和待确认项；
- 当前项目资料和用户确认内容优先于本目录；
- 业务参考不能直接变成 UI 颜色、字体、布局或组件规范；
- 企业 UI 参考使用 Ant Design 5、Carbon Design System、WCAG 2.2 和 `ui-ux-pro-max`。

## 分类

- `lims`：实验室、样品、仪器、检测任务、质量和数据完整性
- `mes`：制造执行、MOM、生产现场、设备和制造数据
- `product-rules`：通用产品说明、状态、流程和异常闭环
- `ui-rules`：通用 UI 参考与验收规则
- 根目录文件：工业域融合、主数据、资产生命周期和数据互操作参考
