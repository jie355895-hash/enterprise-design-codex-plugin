# 工业SaaS与主数据治理参考

## SaaS数据层级

租户 → 公司 → 组织/部门 → 工厂/园区 → 车间/产线/实验室 → 项目/产品/设备/样品/批次。

## 主数据对象

- 组织、工厂、实验室、位置
- 产品、物料、批次和工单
- 设备、仪器、能力和方法
- 样品、检测项、单位和量纲
- 用户、岗位、角色和数据范围
- 供应商、客户和外部协作方

## 设计关注点

- 主数据与业务交易数据分开。
- 跨系统对象必须有唯一标识或映射关系。
- 租户、组织、工厂和实验室数据范围必须影响页面入口、默认视图和可用操作。
- 管理员权限、业务操作权限、流程权限和数据权限不能混为一个角色字段。
- 数据变更应保留来源、修改人、时间、原因和生效版本。

本文件只作为SaaS工业系统分析参考，不直接定义任何项目的权限模型。
