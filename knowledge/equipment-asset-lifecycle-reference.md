# 设备与资产全生命周期参考

## 生命周期

规划/采购 → 到货验收 → 安装/配置 → 投运 → 运行监测 → 点检/维护 → 校准/检定 → 故障/维修 → 变更 → 停用 → 报废/替换 → 履历归档。

## 业务对象

- 资产、设备、仪器、部件和位置
- 设备层级和关键度
- 运行状态和可用状态
- 维护、点检、保养、维修和备件
- 校准/检定证书、有效期和超期状态
- 设备风险、异常、责任人和处理任务
- 设备履历、成本、变更和报废

## 设计关注点

- 资产台账、实时状态、维护任务和校准记录应区分。
- 设备异常应关联责任角色、处理任务、状态和结果回写。
- 设备是否可用于某检测任务，需要结合状态、校准有效期、能力和权限判断。
- 资产管理可与MES、LIMS、IoT和仓储共享对象，但主责必须确认。
