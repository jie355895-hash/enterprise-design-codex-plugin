# 外部业务参考来源登记

本文件记录本次知识库更新使用的外部业务资料。外部资料只作为业务参考，不自动转化为插件强制规则，也不替代当前项目需求和用户确认。

| 领域 | 来源 | 主要借鉴内容 | 使用边界 |
|---|---|---|---|
| LIMS质量 | [WHO LQMS Handbook](https://www.who.int/publications/i/item/9789241548274) | 质量体系、记录、样品、人员、设备、审核和持续改进 | 公共卫生/临床内容需按项目判断 |
| LIMS样品 | [WHO Quality Manual Template](https://www.who.int/publications/m/item/laboratory-quality-manual) | 样品验收、运输、结果验证、关键结果和不符合事件 | 不直接套用表单和权限 |
| 实验室信息化 | [ASTM E1578](https://store.astm.org/e1578-18r26.html) | LIMS、LES、ELN、SDMS、CDS及生命周期 | 不代表某个产品必须采用全部系统 |
| 检测/校准 | [ISO/IEC 17025](https://www.iso.org/standard/66912.html) | 能力、公正性、一致运行和结果可信度 | 仅在检测/校准场景适用 |
| MES | [ISA-95](https://www.isa.org/standards-and-publications/isa-standards) | 层级、MOM活动、对象和系统接口 | 不替代客户现有架构 |
| 电子记录 | [eCFR 21 CFR Part 11](https://www.ecfr.gov/current/title-21/chapter-I/subchapter-A/part-11) | 审计追踪、电子签名、访问、验证和留存 | 仅在适用监管范围内使用 |
| 数据完整性 | [FDA Data Integrity](https://www.fda.gov/regulatory-information/search-fda-guidance-documents/data-integrity-and-compliance-drug-cgmp-questions-and-answers) | 原始数据、审计追踪、记录完整性和调查 | 作为高风险场景参考 |
| GLP | [OECD GLP](https://www.oecd.org/en/topics/sub-issues/testing-of-chemicals/good-laboratory-practice-and-compliance-monitoring.html) | 研究计划、研究负责人、QA、原始数据、报告和归档 | 仅在GLP研究场景适用 |
| 资产管理 | [ISO 55000:2024](https://www.iso.org/cms/%20render/live/en/sites/isoorg/contents/data/standard/08/30/83053.html) | 资产价值、生命周期、风险和持续改进 | 不等同于具体EAM产品方案 |
| 工业互操作 | [OPC UA](https://opcfoundation.org/about/opc-technologies/opc-ua/) | 信息模型、事件、历史数据、安全和互操作 | 不强制指定协议 |
