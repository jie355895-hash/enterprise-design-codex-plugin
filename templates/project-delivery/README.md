# 项目交付目录

每个产品设计项目统一使用以下结构：

```text
项目目录
├─ Demo/                         Demo 入口和前端资源
├─ design-confirmation.md        Demo 前文字确认稿
├─ role-permission-matrix.md     角色、数据和权限矩阵
├─ assumptions.md                默认假设与待确认项
├─ demo-review.md                Demo 生成后评审结果
├─ version.json                  当前版本记录
└─ PROJECT_CONTEXT.md            项目上下文
```

`demo/` 内可继续按项目技术栈组织 `index.html`、`src/`、`assets/` 等文件。
