# Demo 生成后检查清单

- [ ] 页面结构与已确认PRD一致
- [ ] UI视觉来源和项目覆盖关系已记录
- [ ] 默认1920×1080或PRD指定画布已通过检查
- [ ] 适用桌面宽度下没有布局溢出或内容遮挡
- [ ] 对比度、键盘焦点、焦点遮挡和交互目标符合UI验收基线
- [ ] Demo已记录PRD文件路径、PRD版本和来源关系
- [ ] Demo内说明已根据PRD详细生成
- [ ] Demo内说明没有编造PRD未提供依据的业务规则
- [ ] 不同角色的入口、数据、内容和操作确实有差异或已明确说明无差异
- [ ] 角色、权限、数据范围和流程权限有来源依据
- [ ] 桌面端和窄屏下无遮挡、溢出或截断
- [ ] 加载、空数据、错误、无权限和成功状态完整
- [ ] 业务图标风格统一，说明图标仅使用统一的圆形 `i`
- [ ] 说明入口使用右侧 Drawer，并根据业务复杂度、功能、数据、角色、状态、权限、流程和风险决定说明范围
- [ ] 只有业务含义、功能、数据、交互、角色、状态和权限基本一致的重复组件才合并说明；同一组件承载不同业务功能时已分别说明
- [ ] 业务按钮没有被说明功能替代
- [ ] 需处理提醒包含去处理入口、状态变化和结果回写
- [ ] Demo 说明、角色矩阵、假设、待确认项和版本记录已补齐
- [ ] 已完成AI产品设计评审
- [ ] 已完成用户查看、点击和体验验收，或已明确记录待用户验收
