## V1.1.1 规范修正

修正全局说明体系的入口规则：

1. 说明 icon 必须统一为小号圆形 `i` / `ⓘ`
2. 说明 icon 只能用于打开右侧 Drawer
3. 说明 icon 不得与业务按钮、筛选项、状态标签混合
4. 同类重复组件不重复说明
5. 卡片、表格行、字段实例等重复对象只保留组件级说明
6. 查看、编辑、流程、下载、更多等业务操作按钮不得绑定说明 Drawer

## V1.1.2 需求澄清修正

修正产品设计任务中的需求澄清规则：

1. 需求澄清不是可跳过步骤
2. 泛化一句话需求不得直接生成最终 Demo
3. Demo 生成前必须判断信息完整度
4. 信息不足时必须先提出最少必要的关键问题
5. 若基于默认假设继续，必须在 Demo 说明和版本记录中写明假设项

## V1.1.3 澄清问题数量修正

修正需求澄清问题数量规则：

1. 不设置固定问题数量
2. 只问当前信息缺口所需的关键问题
3. 不为了满足数量而凑问题
4. 简单需求可以只问一个问题
5. 复杂需求可以分批提问

## V1.1.4 Demo 前文字确认修正

修正 Demo 生成前的确认规则：

1. 没有问题可问时，也不能直接生成 Demo
2. 必须先输出文字版确认稿
3. 文字版确认稿需要覆盖需求理解、页面结构、字段、操作、状态、流程、说明对象和默认假设
4. 用户确认文字版内容无误后，才可进入 Demo 生成
5. 如果仍有信息缺口，则先问最少必要问题

## V1.1.5 角色驱动设计修正

新增角色影响设计规则：

1. 角色不是普通说明字段，必须影响设计判断
2. 不同角色可拥有不同默认视图、字段显隐、高频操作和按钮权限
3. 不同角色可拥有不同数据范围、状态可操作范围和流程节点职责
4. 多角色共用页面必须说明共用内容与角色差异
5. 如果角色缺失且影响结构、字段、按钮、权限、状态或流程，必须先澄清

## V1.1.6 SaaS 工业角色权限修正

增强 SaaS 类工业系统设计规则：

1. 同一个公司、同一个模块、同一个页面，不同角色也可能看到完全不同的数据、操作、页面内容和页面入口
2. 页面设计必须考虑租户、公司、组织、部门、工厂、车间、产线、项目、设备等数据范围
3. 页面入口必须受角色、岗位、数据权限、功能权限和流程权限影响
4. 多角色页面必须在文字版确认稿中输出角色差异矩阵
5. 角色差异矩阵至少覆盖角色、页面入口、数据范围、可见内容、可用操作、默认视图和流程职责

## V1.1.7 角色权限依据修正

修正角色权限推断规则：

1. 不得凭空推断角色能看到哪些页面、数据、操作或入口
2. 角色差异必须来自用户说明、PRD、截图、原型、菜单、权限配置、角色岗位说明、流程规则或数据权限规则
3. 无依据的内容必须标记为待确认或默认假设
4. 角色差异矩阵必须增加依据来源列
5. 如果角色权限影响页面设计且缺少依据，必须先向用户确认

## V1.1.8 Demo 说明沉淀修正

新增 Demo 说明沉淀规则：

1. 角色差异矩阵必须写入 Demo 说明
2. 角色权限依据来源必须写入 Demo 说明
3. 默认假设和待确认项必须写入 Demo 说明
4. 数据范围、页面入口、操作权限、流程权限假设必须写入 Demo 说明
5. 版本记录必须同步记录依据、假设和待确认项

## V1.1.9 产品形态分流修正

新增 SaaS 工业系统产品形态判断规则：

1. SaaS 工业系统不只包含企业后台
2. 支持工作台、数据大屏、LIMS、MES、IoT、数字孪生、质量/检测、试验管理、设备/资产等形态
3. 不得默认所有需求都是列表+详情后台页面
4. 设计前必须判断产品形态，并在文字版确认稿中写明
5. 新增数据大屏和 LIMS 的优先澄清规则、UI规则和说明内容标准

## V1.1.10 消息提醒规则修正

新增消息提醒设计规则：

1. 消息提醒分为正常提醒和需处理提醒
2. 正常提醒以阅读、知晓、查看详情、标记已读为主
3. 需处理提醒必须提供去处理入口
4. 设备报警、任务临期、任务超期、异常待确认必须形成处理闭环
5. 文字版确认稿和 Demo 说明必须说明提醒类型、触发规则、处理入口、状态流转和依据来源

## V1.1.36 说明范围修正

修正“重复组件只保留组件级说明”的过度简化规则：

1. 说明数量根据业务复杂度、功能、数据含义、角色、状态、权限、流程、风险和易误解程度综合判断
2. 重复视觉组件只有在业务含义、功能、数据、交互、角色、状态和权限基本一致时，才合并为组件级说明
3. 同一组件承载不同业务功能、数据含义、角色、状态、权限、流程或风险场景时，必须分别提供对应说明
4. 新增 Demo 检查、视觉验收、设计确认稿和 Demo 评审记录的说明范围检查项
